# 마피아 게임 기본 룰

이 문서는 오프라인 파티/모임에서 바로 사용할 수 있는 기본 마피아 규칙입니다.

## 1) 게임 목표

- 시민팀 승리: 모든 마피아를 찾아 제거하면 승리합니다.
- 마피아팀 승리: 생존한 마피아 수가 생존한 시민(비마피아) 수와 같아지면 승리합니다.

## 2) 역할 구성

### 필수

- 사회자 1명: 진행만 담당, 승패 팀에 속하지 않습니다.
- 마피아: 밤에 제거할 대상을 선택합니다.
- 시민: 낮 토론과 투표로 마피아를 찾아냅니다.

### 선택(추천)

- 경찰: 밤에 1명을 조사해 마피아 여부를 확인합니다.
- 의사: 밤에 1명을 보호해 마피아 공격을 막을 수 있습니다.

## 3) 인원별 추천 예시

| 총 인원(사회자 제외) | 추천 구성 |
| --- | --- |
| 6명 | 마피아 1, 경찰 1, 의사 1, 시민 3 |
| 8명 | 마피아 2, 경찰 1, 의사 1, 시민 4 |
| 10명 | 마피아 2, 경찰 1, 의사 1, 시민 6 |

## 4) 진행 순서

1. 역할 배정
   - 사회자가 무작위로 역할을 배정합니다.
2. 밤
   - 모두 눈을 감습니다.
   - 사회자 지시에 따라 순서대로 행동합니다.
     - 마피아: 제거 대상 1명 선택
     - 경찰: 조사 대상 1명 선택
     - 의사: 보호 대상 1명 선택
3. 낮
   - 사회자가 밤 결과(사망자 유무)만 공개합니다.
   - 생존자 전원이 토론합니다.
   - 투표로 1명을 지목해 처형합니다.
4. 반복
   - 밤/낮을 승리 조건이 충족될 때까지 반복합니다.

## 5) 투표 및 판정 규칙

- 기본적으로 최다 득표 1명을 처형합니다.
- 동점이면 아래 중 하나를 고정 룰로 사용합니다.
  - 재투표 1회 후 다시 동점이면 무처형
  - 즉시 무처형
- 사망자는 원칙적으로 토론/투표에 참여하지 않습니다.

## 6) 역할 충돌 처리

- 마피아의 공격 대상이 의사의 보호 대상과 같으면 해당 밤 사망자는 없습니다.
- 경찰 조사 결과는 경찰 본인만 확인합니다.
- 의사의 자기 보호 가능 여부는 시작 전에 고정합니다.

## 7) 빠른 진행용 옵션(선택)

- 낮 토론 시간 제한: 2~3분
- 최종 변론 시간: 지목된 사람당 30초
- 게임 시간 단축이 필요하면 재투표 없이 동점 무처형을 사용

## 8) 시작 전 합의 체크리스트

- 동점 처리 방식
- 의사 자기 보호 가능 여부
- 유언(사망 직후 마지막 발언) 허용 여부
- 낮 토론 시간 제한

위 4가지를 먼저 합의하면 진행이 훨씬 매끄럽습니다.
